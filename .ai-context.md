# AI Context: SmartMemo 프로젝트

## 프로젝트 요약
SmartMemo는 AI 기반 스마트 메모 앱입니다. 사용자가 메모를 작성하면 AI가 자동으로 태그를 추천하고, 긴 메모는 요약을 생성합니다. LocalStorage를 사용하여 별도 백엔드 없이 독립 동작합니다.

## 기술 스택
- **프론트엔드**: React 18, TypeScript 5, Vite 5
- **스타일링**: TailwindCSS 3
- **아이콘**: lucide-react
- **데이터 저장**: LocalStorage (JSON 직렬화)
- **AI (선택)**: Claude API (없어도 폴백 로직으로 완전 동작)

## 핵심 설계 결정

### 1. AI 폴백 패턴
AI API 키가 없을 때도 앱이 완전히 동작해야 합니다.
```
AI 있음 → Claude API로 태그 추천 + 요약 생성
AI 없음 → 키워드 빈도 기반 태그 추출 + 본문 앞부분 자르기
```
이 패턴은 `hooks/useAI.ts`에서 관리합니다.

### 2. 상태 관리
별도 상태 관리 라이브러리(Redux 등) 없이 React의 useState + useReducer + Context로 구현합니다.
- `useMemos` 훅: 메모 CRUD + LocalStorage 동기화
- `useSearch` 훅: 검색어 디바운스 + 필터링 로직
- `useAI` 훅: AI 호출 + 폴백 로직

### 3. 데이터 구조
```typescript
// 메모 하나의 타입
interface Memo {
  id: string;          // crypto.randomUUID()
  title: string;
  content: string;
  summary: string;     // AI 요약 or 앞 100자
  tags: string[];      // AI 추천 or 키워드 추출
  isPinned: boolean;
  createdAt: string;   // ISO 8601
  updatedAt: string;   // ISO 8601
}
```
LocalStorage 키: `smartmemo_data`

### 4. 컴포넌트 계층
```
App
├── Layout
│   ├── Header (로고, 검색바, 새메모 버튼)
│   ├── Sidebar (태그 필터)
│   └── Main
│       └── MemoList
│           └── MemoCard (개별 메모)
├── MemoEditor (모달)
└── ConfirmDialog (삭제 확인)
```

### 5. 반응형 브레이크포인트
```
- sm (< 768px): 사이드바 숨김, 1열 레이아웃
- md (768px ~ 1024px): 사이드바 접힘, 2열 그리드
- lg (> 1024px): 사이드바 펼침, 3열 그리드
```

## 코딩 규칙
- 모든 컴포넌트는 함수형 (FC) + TypeScript 타입 명시
- Props 인터페이스는 컴포넌트 파일 상단에 정의
- 커스텀 훅은 `use` 접두사 사용
- 이벤트 핸들러는 `handle` 접두사 사용
- 유틸 함수는 순수 함수로 작성
- TailwindCSS 클래스는 인라인으로 사용 (별도 CSS 최소화)
